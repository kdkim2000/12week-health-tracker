# 12주 건강관리 체크리스트 앱

12주간의 운동습관과 생활지표를 기록하고 관리할 수 있는 모바일 웹 애플리케이션입니다.

## 🌟 주요 기능

### 1. 사용자 인증
- 간단한 이메일/비밀번호 기반 로그인/회원가입
- 로컬 스토리지를 사용한 오프라인 데이터 저장
- 회원가입 시 자동으로 12주 프로그램 시작

### 2. 일일 체크리스트
매일 3가지 항목을 기록:
- ✅ 운동 완료 여부
- 🍎 식단 준수 여부
- ⚖️ 체중 기록 (선택사항)

### 3. 시각화 대시보드
- 📅 12주(84일) 전체를 한눈에 보는 달력
- 📊 주차별 달성률 통계
- 📈 전체 진행률 표시
- 🎨 색상 코딩:
  - 초록색: 모든 항목 완료
  - 노란색: 일부 항목 완료
  - 빨간색: 미완료
  - 회색: 미래 날짜

### 4. 모바일 최적화
- 반응형 디자인 (모바일 우선)
- 터치 친화적 UI
- 빠른 로딩 속도

## 🛠️ 기술 스택

- **Framework**: Next.js 15.x (App Router)
- **UI Library**: React 19.x
- **Language**: TypeScript
- **UI Components**: Material-UI (MUI) v7.0
- **Storage**: 브라우저 로컬 스토리지

## 📦 설치 방법

### 1. 프로젝트 클론 또는 파일 복사
```bash
# 프로젝트 디렉토리 생성
mkdir 12week-health-tracker
cd 12week-health-tracker

# 모든 파일을 해당 디렉토리에 복사
```

### 2. 의존성 설치
```bash
npm install
```

### 3. 개발 서버 실행
```bash
npm run dev
```

브라우저에서 http://localhost:3000 접속

## 🚀 사용 방법

### 1. 회원가입
1. 첫 화면에서 "회원가입" 클릭
2. 이메일과 비밀번호 입력 (비밀번호는 6자 이상)
3. 회원가입 완료 시 자동으로 12주 프로그램 시작

### 2. 일일 기록
1. 메인 대시보드에서 달력의 날짜 클릭
2. 운동 완료, 식단 준수 체크
3. 체중 입력 (선택사항)
4. 저장 버튼 클릭

### 3. 진행 상황 확인
- 상단의 진행률 바에서 전체 달성도 확인
- 달력에서 각 날짜의 색상으로 달성 여부 확인
- 주차별 통계 카드에서 상세 정보 확인

## 💾 데이터 저장

이 앱은 브라우저의 **로컬 스토리지**를 사용하여 데이터를 저장합니다:

### 장점
- ✅ 서버 없이 동작
- ✅ 오프라인 지원
- ✅ 빠른 성능

### 주의사항
- ⚠️ 브라우저 캐시를 지우면 데이터가 삭제됩니다
- ⚠️ 다른 기기나 브라우저에서는 데이터 동기화 안 됨
- ⚠️ 실제 서비스에서는 백엔드 서버와 데이터베이스 필요

## 🔒 보안 고려사항

현재 구현은 **학습 목적**입니다. 실제 서비스에서는 다음 사항들을 추가해야 합니다:

1. **비밀번호 암호화**: bcrypt 등을 사용한 해시 처리
2. **HTTPS**: SSL/TLS 인증서 적용
3. **백엔드 API**: 서버 사이드 인증 구현
4. **데이터베이스**: 안전한 데이터 저장소
5. **입력값 검증**: 서버 사이드 유효성 검사
